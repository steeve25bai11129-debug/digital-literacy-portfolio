## How to Run / View This Project

This project does not require any code installation. All deliverables are document-based. Follow these steps:

### Step 1 – Clone the Repository
```bash
git clone https://github.com/Steeverex/digital-literacy-portfolio.git
cd digital-literacy-portfolio
```

### Step 2 – View Task Files
```bash
# List all files
ls -R

# Read any markdown file
cat task-4-email-etiquette/social-media-checklist.md
cat task-5-cybercrime/casestudy.md
cat task-5-cybercrime/prevention-checklist.md
```

### Step 3 – View the Project Report
Open `report/project-report.pdf` or `report/project-report.docx` in any document viewer.

### Step 4 – View Infographic and Screenshots
Open the image files inside `task-1-presentation/` and `task-2-portfolio/` and `task-3-platforms/`.
